---安裝環境---
1. 安裝 python 3

2. 使用 conda 創一個虛擬環境
    a. 安裝 conda
    b. 開啟 cmd 輸入 conda create -n human python=3.6 ，等他跑完
    
3. 開啟 cmd，cd 到這個專案資料夾

4. 啟動虛擬環境
    a. 在 cmd 輸入 activate human
    
5. 輸入 pip install -r REQUIREMENTS.txt，開始安裝所需要的套件

6. 完成

pg_ctl start -D "C:\Program Files\PostgreSQL\11\data"
